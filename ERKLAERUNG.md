# ProfilePulse — Einfach erklärt

## Was ist Malt?

Malt ist eine Online-Plattform, auf der Unternehmen Freelancer (also Selbstständige) finden und buchen können — ähnlich wie eine Jobbörse, aber speziell für freie Mitarbeiter. Tausende Freelancer aus IT, Design, Marketing und anderen Bereichen haben dort ein Profil.

## Das Problem

Malt verlangt von jedem Freelancer, dass er **alle 7 Tage** auf einen Button klickt, um zu bestätigen: "Ja, ich bin noch verfügbar für neue Aufträge."

Wenn man das vergisst:
- Verschwindet ein grünes Badge ("Verfügbarkeit bestätigt") vom Profil
- Wird man in den Suchergebnissen weiter nach unten sortiert
- Sehen potenzielle Kunden das Profil seltener
- Bekommt man weniger Anfragen — also weniger Umsatz

Das ist so, als müsste man jede Woche ein Schild an seiner Ladentür umdrehen, sonst wird der Laden auf Google Maps schlechter angezeigt.

## Die Lösung

Dieses kleine Programm erledigt das **automatisch**. Einmal einrichten, danach läuft es jeden Tag von alleine im Hintergrund:

1. Öffnet unsichtbar einen Browser
2. Geht auf die Malt-Website
3. Klickt auf "Ja, ich bin verfügbar"
4. Klickt auf "Bestätigen"
5. Schließt den Browser wieder

Der Freelancer muss nie wieder daran denken.

## Die Geschäftsidee

**Zielgruppe:** Alle Freelancer auf Malt.de — das sind zehntausende Menschen allein im deutschsprachigen Raum.

**Warum würden Leute dafür zahlen?**
- Jeder vergessene Klick kann bedeuten, dass man einen Auftrag über mehrere tausend Euro verpasst
- Die meisten Freelancer sind technisch versiert genug, um den Wert zu verstehen
- Es spart jede Woche Zeit und mentale Energie ("Habe ich diese Woche schon bestätigt?")

**Preismodell:**
- **Monats-Abo:** 5 EUR/Monat — jederzeit kündbar, ideal zum Ausprobieren
- **Lifetime-Lizenz:** 49 EUR einmalig — einmal zahlen, für immer nutzen

Die Bezahlung läuft über **Stripe** (sichere Kreditkarten-/Kartenzahlung). Nach dem Kauf erhält man einen Lizenzschlüssel, den man im Tool aktiviert. Der Schlüssel wird gegen einen Lizenz-Server geprüft.

**Warum das funktionieren kann:**
- Es gibt kein vergleichbares Tool auf dem Markt
- Die Plattform Malt wächst stark in Europa
- Das gleiche Prinzip lässt sich auf andere Freelancer-Plattformen übertragen (Freelancermap, Upwork, etc.)
- Bei einem Stundensatz von 80-150 EUR rechnet sich das Tool schon beim ersten gewonnenen Auftrag

## In einem Satz

> Ein kleines Programm, das Freelancern automatisch dabei hilft, auf einer Jobbörse sichtbar zu bleiben — damit sie mehr Aufträge bekommen, ohne jede Woche daran denken zu müssen.
